# README

Sample Rails app to make HTTP requests using REST client with a script file.

## To run this script use: 

```
$ ruby app/script/rest_requestor.rb
```

## Ruby version:

```
$ ruby-2.5.1
```

## Rails version:

```
$ rails-5.2.3
```
